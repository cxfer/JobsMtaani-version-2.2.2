JobsMtaani Setup

Welcome to JobsMtaani service booking platform

Setup Instructions
  1. Make sure your MySQL server is running
  2. Create a database named jobsmtaani
  3. Update database credentials in config/database.php
  4. Click the button below to install the database tables
Install Database
Database Configuration

Make sure to update the following in config/database.php: 

$host = 'localhost';
$dbname = 'jobsmtaani';
$username = 'your_username';
$password = 'your_password';
